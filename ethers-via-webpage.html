This page allows you to interact with the Ethereum blockchain just via your browser, including on mobile devices.
The javascript library "ethers" has been loaded and can be interacted with using this page.
You will need to set up how you intend to connect to the blockchain via ethers commands, such as Snippet 1 or Snippet 2.
If you are on a mobile without a Dapp browser, Snippet 1 will probably not work.

Snippets

1. MetaMask setup (requires window.ethereum, such as from MetaMask for PCs):


window.ethereum.enable();
window.provider = new ethers.providers.Web3Provider(window.ethereum);
window.signer = provider.getSigner();
return "Done";

2. Simple Ethers setup (should work on any browser):


window.provider = new ethers.getDefaultProvider();
return "Done";

3. Seed a random address:


window.signer = ethers.Wallet.createRandom().connect(provider);
return signer.address;

4. Retrieve private key of randomly generated address


return signer.address;

5. Retrieve mnemonic used to generate a randomly generated address


return signer.mnemonic;
